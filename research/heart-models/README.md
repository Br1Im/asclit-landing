# Heart 3D model research — Asclit

Research and assets for choosing a **beautiful + anatomically accurate + commercially-licensable** 3D heart model for the Asclit lesson module (`lesson.html`, §1.2 "Сердце").

> ⚖️ **Licensing constraint:** Asclit is free but has a **paid subscription** → it is a **commercial** product.
> Therefore **CC BY-NC / CC BY-NC-SA models are OFF-LIMITS.** Only **CC BY / CC BY-SA / CC0** are usable.

---

## TL;DR / recommendation

| Goal | Pick | License |
|------|------|---------|
| Real per-part click/hover highlighting | **BodyParts3D / Z-Anatomy** (segmented) | CC BY-SA |
| Clinically validated, free | **NIH 3D Heart Library** | CC0 |
| Prettiest single-mesh, legal | "Anatomically Correct Human Heart" (Pigcraft, Sketchfab) | CC BY |

**Chosen base for production:** the segmented **BodyParts3D** model in `models/` (CC BY-SA), which gives each
anatomical structure as a separate named mesh → enables true highlighting. A textured/“beautiful” render can
be reproduced from this clean source legally.

---

## Current site model (the problem)

`/heart_new.glb` (used by `index.html` + `lesson.html`) is a **single fused mesh** generated by **Tripo AI**
(1 material, node `tripo_node_…`). It cannot be split, so the lesson "pins" (Aorta / Truncus pulmonalis /
Ventr. sinister / Apex cordis) are just **flat 2D labels at fixed screen %** — they do not anchor to the 3D
geometry and don't rotate with the model. For real highlighting we need a **segmented** model.

## Models evaluated

### ✅ Usable (commercial-OK)
- **BodyParts3D / Anatomography** — DBCLS, Univ. of Tokyo. Built from real full-body MRI; used as university
  lecture material. Per-structure meshes. **CC BY-SA 2.1 JP.** → *the source we downloaded (`models/`).*
- **Z-Anatomy** — community atlas built on BodyParts3D, structures named in Latin (matches `lesson.html`).
  **CC BY-SA 4.0.**
- **NIH 3D Heart Library / 3D Print Exchange** — clinically accurate, reconstructed from patient scans.
  **CC0.** (Often whole-surface, not split by structure.)
- **Visible Human Project (NLM)** — public domain, but raw cryosection/CT/MRI → needs reconstruction.
- "Anatomically Correct Human Heart" (Pigcraft), "Realistic Human Heart" (neshallads) — pretty single-mesh,
  **CC BY** (see `comparisons/all_hearts.png`).

### ⛔ Validated but NOT usable (CC BY-NC-SA — preview only)
University-quality but non-commercial, so they **cannot** ship in Asclit:
- **Univ. of Dundee** — normal heart + separate coronary-arteries model.
- **Leiden / TU Delft / Groningen** — heart + lungs with numbered labels.
- **Univ. Maastricht** — plastinated-specimen photogrammetry.
- **Univ. of Groningen "Anatomy of the heart"** (E-learning UMCG) — the prettiest "textured #1".
  Sketchfab: `sketchfab.com/3d-models/cardiac-anatomy-external-view-e8e8c6685e82474d8393693fd0646dc7`
  - ⚠️ **License conflict:** the model's baked watermark says *"licensed under CC BY-SA 4.0"*, but the
    Sketchfab page sets **CC BY-NC-SA**. It is a derivative of BodyParts3D (CC BY-SA). Because of the
    contradiction we treat it as **NC / risky** and instead reproduce its look from the clean BodyParts3D
    source — which is unambiguously CC BY-SA.

## What's in this folder

```
models/
  heart_segmented.glb   # BodyParts3D heart, structures as named meshes (CC BY-SA)
  heart_web.glb         # web-optimized variant
  *.obj                 # individual structures (see mapping below)
  heart_viewer.html     # standalone Three.js viewer for the GLB
  parts_list_e.txt      # FMA id → English name reference (BodyParts3D)
comparisons/
  all_hearts.png        # contact sheet of the open (non-NC) candidate models
scripts/
  build_glb.py / build_heart.py   # assemble OBJ structures → segmented GLB
  render_glb.py                   # render multi-angle previews of a GLB
  find_models.py                  # Sketchfab search helper
```
